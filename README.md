# FFmpeg EncodeTools GUI

基于 FFmpeg 的，更适配字幕组工作流程的压制工具箱

---

### 写在前面：
>这个软件的开发始于 2026 年 2 月初。最初是因为字幕组群里有人需要用到 FFmpeg 压制视频，但 FFmpeg 本身是个命令行工具，对新手有一定门槛；而市面上的相关 GUI 要么集成了太多不必要的功能，要么和字幕组的日常使用场景脱节。于是我便想做一个 .bat 脚本，大家按引导输入 CRF 和 Preset 等参数，就能一键完成简单的压制。

>起初 bat 脚本的可玩性太低，我就转向了 Python。恰好组内还有清洗字幕的需求，便又加入了字幕清洗功能；几天后，又有人提出想要字幕封装。就这样，软件越做越大，功能也越加越多。

>到 0.4.7 版本结束之后，我觉得想继续拓展功能，就必须从 CLI 转向 GUI 开发。于是试着做了第一版图形界面，随后便投入了长达数月的开发。如今，GUI 版本终于趋于稳定，基础功能也打磨得差不多了。非常感谢每一位朋友对软件的测试和反馈。

>今后的开发计划，将向更适合字幕组的压制工具特化。


## 核心功能

### 1. 视频压制
- 支持配置 FFmpeg 自带的常用编码器（如 libx264、libx265、libsvtav1 ）
- 支持烧录外部 ASS / SRT 字幕文件
- 可指定外部字体文件夹以避免不必要的字体安装。

### 2. 混流封装
- 支持对多个视频轨、音频轨、字幕轨及章节数据进行合并封装。
- 采用不同色块区分不同文件。
- 支持编辑音轨的语言标记与轨道显示名称。
- 针对 MKV 封装，支持将外部字体文件作为附件直接打包入视频文件中。

### 3. MP4 快速封装
- 基于 MP4box 或 FFmpeg，在使用 x264、x265 编码器压制时候后，可快速封装成 MP4 文件。
- 支持在封装过程中同步写入章节信息。

### 4. 字幕文本清洗
- 支持读取并解析 ASS / SRT 格式的字幕文件。
- 自动过滤字幕文件中的特效样式代码，提取纯净的对白文本。
- 支持按行数将提取出的对白文本等份切分为多个子文件。

### 5. 章节封装
- 解析 TXT 格式章节文件，自动转换为符合 FFmpeg 标准的章节元数据，并写入输出视频中。

### 6. VapourSynth 压制
- 支持加载 VapourSynth 脚本（.vpy）。
- 将 VSpipe 输出的 Y4M 原始视频流传输给 x265/x264 编码器或 FFmpeg。
- 压制过程不需要生成大容量中间临时视频文件，节省硬盘空间。

### 7. 任务队列
- 提供任务队列，适配批量压制，支持任务的新增、删除和优先级调整。
- 可配置任务执行完毕后的动作：发送系统通知、打开输出文件夹、退出软件或延时关机。

## 安装与运行

### 运行环境

- 操作系统：Windows 10 或更高版本
- Python 版本：Python 3.9 或更高版本

### 1. 安装 Python 依赖

```bash
pip install PySide6 PySide6-Fluent-Widgets PySide6-Frameless-Window
```

### 2. 准备外部工具依赖
为了确保所有压制、封装和分析功能正常运行，建议在 `项目目录` 目录下新建一个 `工具箱` 文件夹，并放置对应的外部可执行程序，也可以在软件启动后在“系统设置”页面手动指定以下程序的路径：

- `ffmpeg.exe`
- `ffprobe.exe`
- `mp4box.exe`
- `Vapoursynth/vspipe.exe`
- `x265.exe` / `x264.exe`

**推荐使用以下文件夹结构：**
```
FFmpegEncodeToolsGUI/
├── EncodeTools/
│   ├── ffmpeg.exe
│   ├── ffprobe.exe
│   ├── mp4box.exe
│   ├── x265.exe
│   └── Vapoursynth/
│       └── vspipe.exe
├── config/
├── core/
├── ui/
├── utils/
└── main.py
```

### 3. 运行主程序
```bash
python main.py
